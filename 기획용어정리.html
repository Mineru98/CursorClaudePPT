기획 용어 정리

웹 기획자를 위한 필수 용어 가이드

미네루

개요

  • 기본 용어 (UI/UX, 인터페이스, 인터렉션)
  • 기획 단계 용어 (벤치마킹, 와이어프레임, 스토리보드)
  • UI 관련 용어 (팝업, 드롭다운, 토글 등)
  • 웹사이트 구성 용어 (메인 페이지, GNB, 푸터 등)
  • 실무 용어 (구축, 유지보수, 반응형 웹 등)

기본 용어

UI (User Interface)

사용자가 제품/서비스를 사용할 때 마주하는 시각적인 디자인 화면

UX (User Experience)

사용자가 만족하고 편리하도록 사용자 중심적으로 설계된 디자인

인터페이스

사람과 컴퓨터, 시스템, 기기, 도구 등 그 사이에서 일어나는 상호작용을 매개하는 장치

인터렉션

사용자와 제품/서비스의 상호작용

기획 단계 용어

벤치마킹

경쟁사나 특정 분야에서 우수한 상대를 표적으로 성과 차이를 비교하며 분석

와이어프레임

서비스 레이아웃을 구상하는 단계에서 손으로 그리거나 PPT로 간단하게 그린 그림

스토리보드

페이지 구성과 기능들을 화면의 흐름에 따라 표현한 최종적인 산출 문서

프로토타입

설계가 끝난 화면이 어떻게 구동되는지 직접 실행해 볼 수 있는 데모

플로우 차트

흐름을 특정한 순서도 기호를 사용해서 도식적으로 표현한 다이어그램

페르소나

제품이나 서비스를 사용할 만한 목표 연구 집단 안에 있는 다양한 사용자 유형을 대표하는 가상의 인물

UI 관련 용어

팝업

새 브라우저 창이 띄워지는 일반 팝업

모달

현재 머무는 브라우저 창에 띄워지는 레이어 팝업

토스트 팝업

몇 초간 나타났다가 사라지는 팝업 메세지

드롭다운 메뉴

메뉴 제목이 표시된 곳을 선택하면 메뉴가 아래로 펼쳐지는 스타일

토글

on/off를 설정할 때 쓰이는 컨트롤

라디오버튼

중복 선택 불가능한 동그랗게 생긴 버튼형 선택 수단

체크 박스

중복 선택이 가능한 선택 수단

툴팁

마우스를 호버할 경우 1~2초 뒤에 메뉴에 대해 말풍선으로 뜨는 설명

페이지네이션

페이지의 일련번호를 정하는 것

웹사이트 구성 용어

Header (Top)

GNB (Global Navigation Bar)

메인 화면 상단에 위치한 주 메뉴바

Visual

GNB 하단의 디자인적 요소가 가미된 이미지 / 영역

Contents

사용자에게 제공되는 모든 정보 및 이미지

Footer

홈페이지 최하단에 고정되어있는 영역 (주로 사업자 번호, 전화번호 등의 회사 정보)

실무 용어 사용 빈도

결론

기획 용어는 분야가 광범위하며 지속적인 학습이 필요합니다.

용어를 정확히 이해하면 커뮤니케이션 효율이 높아집니다.

실무에서는 용어의 정확한 사용이 프로젝트 성공의 열쇠입니다.

자주 묻는 질문

UI/UX, 와이어프레임, 스토리보드, 프로토타입 등의 기본 용어를 먼저 숙지하는 것이 좋습니다. 이러한 용어들은 기획 과정에서 가장 빈번하게 사용됩니다.

실제 프로젝트에 참여하거나, 기획 관련 서적을 읽거나, 온라인 커뮤니티에 참여하는 것이 좋습니다. 또한 다양한 웹사이트나 앱을 분석하며 용어를 실제로 적용해보는 연습도 효과적입니다.

스토리보드, 와이어프레임, 프로토타입과 같은 산출물 관련 용어와 함께 기능 명세에 관련된 용어들이 중요합니다. 또한 API, 백엔드, 프론트엔드와 같은 기본적인 개발 용어도 이해하고 있으면 소통에 도움이 됩니다.
