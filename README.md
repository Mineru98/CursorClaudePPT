## 매직 프롬프트

### 가이드

```
다음 내용을 기반으로 프로페셔널한 프레젠테이션 자료를 작성해주세요:
 
【자료 유형】

- 프레젠테이션 형식의 제안서
- 영업 자료
- 기획서
- 보고서
- 기술 세미나 자료

【디자인 지정】
 
16:9 화면 비율
모던하고 시각적으로 매력적인 디자인
기업 / 컨설팅 수준의 완성도
컬러 테마: 연보라색 계열 // RGB 값으로 색상을 지정하거나, 푸른색, 연보라색 등 색을 말해도 좋습니다.
인터랙티브 요소 포함

【기술 사양】
 
tailwind css로 구현
슬라이드 전환 기능
내비게이션 컨트롤
반응형(Responsive) 디자인

【자료 구성】
 
타이틀 슬라이드
개요 / 요약
주요 포인트 (글머리 기호 또는 도표 활용)
상세 정보 (표 및 그래프 포함)
결론 / 제안 / CTA(Call to Action)
필요에 따라 FAQ 섹션 추가 가능
```
